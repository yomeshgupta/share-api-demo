Navigator Share API Demo

Web is more powerful than ever and it has came a long way! We can do many things which we could have never imagined before. Web Share API went under the radar. It allows developers to add sharing functionality into websites by taking advantage of the native content sharing capabilities on a user‚Äôs device. 

var data = {
    title: 'Code by Devtools.tech',
    text: 'Ready to test your knowledge? Move closer to your dream job?',
    url: 'https://code.devtools.tech'
};

if (navigator.share) {
    navigator
        .share(data)
        .then(() => console.log("Operation Successful"))
        .then(() => console.log("Operation failed"))
} else {
    console.log("Share API not supported!")
}
